Who sent this s#it?

What is this?

Welcome to "Who sent this s#it?", a little game where you guess who sent random funny/weird messages from your group chat. Currently, only WhatsApp chat data is supported. Wanna run this on your own server? Follow the instructions on the GitHub repo. The way it works is simple:

  1. Export your WhatsApp chat data from an English-speaking phone.
  2. Download and install Ollama.
  3. Use a local AI to determine which messages are funny. The full code is available here

                            if score > 3 and DEBUG:

                        except Exception as e:
                            tqdm.write(f"Error: {e} - {response}")
